# Super Store Sales Dashboard Using Power BI

## Project Overview
This project presents an interactive Super Store Sales Dashboard developed using Power BI. The dashboard analyzes sales and profit performance across regions, states, customer segments, product categories, sub-categories, and shipping modes.

## Tools and Technologies
- Power BI
- Data Visualization
- Interactive Dashboard Design

## Key Performance Indicators
- Total Sales: 1.57M
- Total Profit: 175.26K

## Dashboard Analysis
- Region-wise sales analysis
- Segment-wise sales analysis
- Monthly sales comparison by year
- Monthly profit comparison by year
- State-wise sales and profit analysis
- Category-wise sales analysis
- Sub-category analysis
- Ship mode analysis

## Dashboard Features
- Interactive region slicer
- KPI cards for Sales and Profit
- Donut chart for Sales by Region
- Donut chart for Sales by Segment
- Year-over-Year monthly sales trend
- Year-over-Year monthly profit trend
- Geographic map for State-wise Profit and Sales
- Bar charts for Category, Sub-Category, and Ship Mode analysis

## Key Insights
- Total sales reached approximately 1.57M
- Total profit reached approximately 175.26K
- The West region contributed the highest share of sales
- Consumer segment generated the highest sales contribution
- Office Supplies recorded the highest category-wise sales
- Monthly sales and profit patterns varied across 2019 and 2020
- State-level performance differences were identified using geographic analysis
- Shipping mode and sub-category performance were compared visually

## Conclusion
This project demonstrates the use of Power BI to build an interactive business intelligence dashboard for analyzing sales, profit, regional performance, customer segments, product categories, shipping modes, and year-over-year trends.
